# Frontend - Ứng dụng Chat Next.js

Frontend Next.js 14 với TypeScript và Tailwind CSS.

## Thiết lập

1. Cài đặt dependencies:
```bash
npm install
```

2. Tạo file `.env.local`:
```env
NEXT_PUBLIC_SOCKET_URL=http://localhost:3001
```

3. Chạy development server:
```bash
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000) trong trình duyệt của bạn.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Socket.io Client
